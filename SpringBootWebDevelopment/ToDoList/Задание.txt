Задание. Создание программы управления задачами
Что нужно сделать
Создайте новый проект ToDoList в папке SpringBootWebDevelopment.
Реализуйте в проекте контроллер так, чтобы при входе через браузер по адресу http://localhost:8080/ на страницу выводились текущие дата и время или случайное число (на ваш выбор). 
Проверьте, что ваше приложение запускается и работает. Познакомьтесь с видео по теме. 
Подключите к проекту библиотеку Hibernate и создайте в ней сущность Task для работы с задачами, содержащую следующие поля:
id;
creationTime — дата и время создания задачи (рекомендуем использовать класс LocalDateTime);
isDone — является ли задача выполненной;
title — название задачи;
description — описание задачи.

Проверьте реализованные вами методы, используя любой инструмент для тестирования HTTP-запросов:
расширение для браузера Chrome Talend API или Postman;
приложение Postman (видео, как пользоваться приложением).
Если вы используете Postman, то в запросе нужно установить следующие параметры:
метод запроса — POST;
путь, по которому будет происходить запрос;
тип запроса — Body, raw и JSON;
тело запроса — в формате JSON.
